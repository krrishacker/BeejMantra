## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd beej-mantra-frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

## Customization

### Adding New Languages

1. Add language definition in `src/data/languages.ts`
2. Add translations in the `translations` object
3. Update the language switcher components

### Modifying Features

Edit `src/data/features.ts` to add, remove, or modify feature cards.

### Styling

The application uses Tailwind CSS. Custom styles can be added in `src/index.css` using the `@layer` directive.
